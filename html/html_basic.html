Tài liệu HTML

Tất cả tài liệu HTML phải bắt đầu bằng khai báo loại tài liệu: <!DOCTYPE html>.

Bản thân tài liệu HTML bắt đầu bằng <html>và kết thúc bằng </html>.

Phần hiển thị của tài liệu HTML nằm giữa <body>và </body>.

Ví dụ

<!DOCTYPE html>
<html >
<body >

<h1 > My First Heading </h1 >
<p > My first paragraph. </p >

</body >
</html >

Tuyên bố <!DOCTYPE>

Khai báo này <!DOCTYPE>>thể hiện loại tài liệu và giúp trình duyệt hiển thị các trang web một cách chính xác.

Nó chỉ được xuất hiện một lần, ở đầu trang (trước bất kỳ thẻ HTML nào).

Tuyên bố này <!DOCTYPE>không phân biệt chữ hoa chữ thường.

Khai <!DOCTYPE>báo cho HTML5 là:

<!DOCTYPE html>

Tiêu đề HTML

Tiêu đề HTML được định nghĩa bằng thẻ <h1>to <h6>.

<h1>định nghĩa tiêu đề quan trọng nhất. <h6>định nghĩa tiêu đề ít quan trọng nhất: 

Ví dụ

<h1 > This is heading 1 </h1 >
<h2 > This is heading 2 </h2 >
<h3 > This is heading 3 </h3 >

Đoạn văn HTML

Các đoạn văn HTML được định nghĩa bằng <p>thẻ:

Ví dụ

<p > This is a paragraph. </p >
<p > This is another paragraph. </p >

Liên kết HTML

Liên kết HTML được định nghĩa bằng <a>thẻ:

Ví dụ

<a href ="https://lamtrong.github.io/laptrinhweb" > This is a link </a >
Hãy tự mình thử »

Điểm đến của liên kết được chỉ định trong hrefthuộc tính. 

Thuộc tính được sử dụng để cung cấp thông tin bổ sung về các thành phần HTML.

Bạn sẽ tìm hiểu thêm về các thuộc tính ở chương sau.

Hình ảnh HTML

Hình ảnh HTML được định nghĩa bằng <img>thẻ .

Tệp nguồn ( src), văn bản thay thế ( alt), widthvà heightđược cung cấp dưới dạng thuộc tính:

Ví dụ

<img src ="logo.png" alt ="laptrinhweb" width ="104" height ="142" >
>Hãy tự mình thử »

Cách xem nguồn HTML

Bạn đã bao giờ nhìn thấy một trang web và tự hỏi "Ồ! Họ làm điều đó như thế nào?"

Xem mã nguồn HTML:

Nhấp vào CTRL + U trên trang HTML hoặc nhấp chuột phải vào trang và chọn "Xem Nguồn Trang". Thao tác này sẽ mở một tab mới chứa mã nguồn HTML của trang.

Kiểm tra một phần tử HTML:

Nhấp chuột phải vào một phần tử (hoặc một vùng trống) và chọn "Kiểm tra" để xem các phần tử được tạo thành từ những gì (bạn sẽ thấy cả HTML và CSS). Bạn cũng có thể chỉnh sửa HTML hoặc CSS ngay lập tức trong bảng Phần tử hoặc Kiểu mở ra.
